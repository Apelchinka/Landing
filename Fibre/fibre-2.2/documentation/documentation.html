Requirements

For installing this template you need to make sure that you have the following extensions on your webserver. You will need to contact your webhost if the following requirements are missing:

  • PHP 5.2+;
  • Apache webserver (For the Page Builder Windows servers will probably work, but are not supported)
  • PHP Mail support;
  • Enabling JavaScript in the web browser.

To be able to edit and upload the theme you need also:

  • A text editor like Sublime Text 2, Notepad++ or Brackets
  • An FTP tool to upload the files like Filezilla or Cyberduck

Installation

Unzip the downloaded files to the local disk on your computer. Inside the extracted folder there will be a folder with documentation and a folder with the html templates. Choose pages which you will use in project and copy all the content related to the layout that interests you to your server via FTP client.

In this archive some images that you can see in demo pages were replaced by a placeholder.

Fonts

Fibre uses "Lato" Google WebFont. You can choose other fonts from google directory editing this line of code in the <head>.

<!-- Google Fonts -->
<link rel="stylesheet" href="http://fonts.googleapis.com/css?family=Lato:300,400,700,900,300italic,400italic,700italic,900italic">

If you want to import new font from Google Webfont library, don't forget to change property in CSS style: font-family with correct font name.

<!-- css/style.css -->

body {
    font-family: "Lato", "Helvetica Neue", Helvetica, Arial, sans-serif;
}

Page Builder

As Fibre is an HTML template, it uses a number of .html pages in conjunction with css stylesheets and a number of javascript plugins. The following is a brief overview of the files and a look at some prominent features.

We do our best while making all files for Fibre. Every file (HTML, CSS, JS) that was created by us has clear code, wrapped with comments – all this for your convenience.

HTML Builder is NOT a CMS (content management system). It does not provide a way for end-users to work on live websites and make changes to such websites.

Installation

Installing the HTML Builder is simply a matter uploading the script files to a local server/hosting account. Once uploaded, point your browser to builder's directory and the builder will work.

For example: www.example.com/Builder or localhost/Builder (if you run it locally on your local server environment).

Don't upload the .zip file you downloaded from themeforest directly to your server, first unzip it, then upload the folder "Builder" which is inside it.

Features

Drag & Drop

HTML Builder allows users to drag HTML blocks onto a canvas and drop them where they'd like them to appear. Once dropped onto the canvas, the position can be changed by dragging and dropping as well. Blocks can be delete individually or all blocks on the current page can be deleted in one go.

Editing Content

The HTML Builder script can also be used to edit written content inside each element. HTML Builder uses a rich text editor, allowing to make changes to the written content. Depending on what type of element you're editing, certain options will be disabled in the editor.

Editing Styles

The HTML Builder script also allows for basic CSS editing. To change CSS styles of an element in the builder you need to click to the element you want to change and you can change the styles in the left panel.

Don't upload the .zip file you downloaded from themeforest directly to your server, first unzip it, then upload the folder "Builder" which is inside it.

The blocks of the category "slider" are not able to change directly in the page builder. If you want to use one of the Slider blocks please add the block to your design in the page builder. After exporting your final page you are able to change and adjust the Slider content and Slider adjustments in the corresponding HTML-, CSS (css/style.css) and JS-Files (js/custom.js).

Editing Links

When selected, a "link" tab will be visible in the left panel, allowing you to either choose a page to link to or enter a URL manually as well as setting the inner text of the link element.

When building a one-page website, you might want to link to sections within a single page rather then to a separate page. To do this, simply edit the link and use the second dropdown to choose the block you'd like to link to.
Editing Images

The HTML Builder script allows user to edit images as well. When selected, a "image" tab will be visible in the left panel, allowing you to either upload an image or enter a URL manually.

If you want to change the background images you can do this when you have finished your Website in the Page builder and have exported your final page.

  1. To change the background images you need to open the style.css (e.g. .newsletter-image) and change the link of the image to the name you stated for your image. Please make that you copy your image into the images folder.
  2. You need to open the respective image in Photoshop (you will find the images in the folder "images")
    • Place your new image
    • Press „Save for Web“
    • and save the new image into the images folder of the fibre website and overwrite the old image file.
Editing Videos

HTML Builder allows users to edit Youtube and Vimeo videos as well. Click the video you'd want to edit. This will open the left sidebar panel where you can enter a Youtube video ID or a Vimeo video ID. You can alter between these as you please, the application will detect which platform you're using and update the video accordingly.

Editing Icons

The HTML Builder script allows user to edit icons as well (the script allows you to use all icons available in the Font Awesome collection). After clicking on an icon on the canvas, a "Icon" tab will be visible in the left panel, allowing you to choose any item from the Font Awesome collection.

The icons of "Linea Fonts" are not only editable in color, size and alignment via the page Builder Tool. The icon itself could only changed in the HTML file after exporting your final website.

More Information
Cloning Elements

Select the element which you want to clone and click the clone button in the left panel. Please note that cloned items will be inserted into the canvas right next to the original item. (provided that enough place is given - if not the cloned element will be placed below)

Multiple Pages

The HTML Builder script allows users to create multiple pages at once. To add a new page, simply click the green button labeled "+ Add" just below the current pages list. You can alter the page names as well (please use names without the .html extension, these are added automatically during the export). When exporting the markup, the pages will use the same names. To delete a page, simply click the little delete icon next to the page you'd like to delete.

Responsive Views

HTML Builder allows users to switch between different view modes, this to ensure the resulting page will look good on mobiles, tablets and desktop screens. To toggle between modes, use the three buttons to the left of the main toolbar.

Global Blocks

HTML Builder allows users to configure so-called "global blocks". These are blocks that maintain the same appearance and settings across different pages. This feature comes in handy when configuring navigations and footers for example. Each block on the canvas has a check box labeled "Global". When checked, changes made to this block will be applied to the same block on all pages in your site, assuming those other blocks are marked as "global" as well.

Preview

If you want to generate a preview you can do this with the button on the top right side to the left of the trashcan.

Export

Whenever the user is happy with the final result, the created markup can be exported by clicking the export button. The export feature will grab all the used HTML blocks for each created page and inserts these blocks into a copy of the HTML skeleton file. Once all pages are completed, all the HTML files are bundled into a ZIP archive together with the other external resources like stylesheets, Javascript files and images.

Dark & Light Version

To adjust the color versions in the whole document, please add the class „dark-version“ for the dark website version or „light-version“ for the light version of the website to your main container. 

<!--- Main -->
<div id="page class="dark-version">
    ...
</div>
<!--- /End Main -->

To define the background colors of the sections, please add one of the following classes to the section you want to change:

  • bg-color: colored background
  • bg-color-light: light colored background
  • bg-color-dark: dark colored background
  • bg-white: white backround
  • bg-light: light backround
  • bg-lightgrey: light grey backround
  • bg-silver: silver backround
  • bg-grey: grey backround
  • bg-darkgrey: dark grey backround
  • bg-dark: dark backround
  • bg-black: black backround
<section id="services class="bg-grey">
    ...
</section>

The colors of the sections / containers are also editable. To change this use one of the following classes:

  • text-color: colored text
  • text-dark: dark grey text
  • text-silver: light grey text
  • text-white: white text

Smooth Scroll

For smooth-scroll you need to add the class „smooth-scroll“ to the a-tag. The hyperlink refers to the corresponding id of the relative section. 

<a href="#section" class="smooth-scroll">Link</a>

Back to Top Button

The "Back to Top" link allows users to smoothly scroll back to the top of the page. It's a little detail which enhances navigation experience on website with long pages. You can disable it on your page by uncommenting in your index.html these lines of code at the end of the file:

<a href="#page" class="btn-circle btn-circle-lg btn-color btn-top smooth-scroll">
    <i class="fa fa-angle-up"></i>
</a>

Animation Effects

Fibre uses ScrollReveal-Plugin to trigger animations on scroll. To learn more about ScrollReveal please read the Documentation of the Plugin.

Predefined initialization in the custom.js

<!—- HTML -—>
<div class="reveal-bottom-opacity">
    ...
</div>

// Javascript
window.sr = ScrollReveal();
sr.reveal('.reveal-bottom-opacity', {
    origin: 'bottom',
    distance: '20px',
    duration: 800,
    delay: 0,
    opacity: 0,
    scale: 0,
    easing: 'linear'
});

Youtube / Vimeo Embed

Embedding videos of Youtube or Vimeo (like the example in the fibre video section) will be accomplished through "data-url" attribute. The video will be embedded as recently as a button event has happen to the video container. As long as there was no button presses by viewer there will be shown an thumbnail of the video. 

<!—- HTML -—>
<div id="video-embed" class="video-wrapper" data-url="https://vimeo.com/117570710">
    ...
</div>

// Javascript
Video('#video-embed');

Charts

The Fibre Template uses the Circle Plugin to show animated pie charts. The charts could be defined through the attributes data-value, date-unit und data-percentage.

  • data-value: responsible for text/numbers in the chart
  • date-unit: Unit (subscript) in the chart *
  • data-percentage: Value in percent (here: colored circle (50 = 180°)
To change the chart colors you need to open the style.css (/css). Scroll down to the property ".background-color" of the ".panel.active > .panel-heading"-selector and change the color to what you need.

Youtube Background

Fibre provides the opportunity to embed a youtube-video in background.

Youtube video embedding accomplished through the attribute data-video-id. You only need to provide the video ID which you can find in the url of the video:

https://www.youtube.com/watch?v=_M1A_nQZw0Q

Youtube video is only enabled for Desktop – videos for mobile and tablets are disabled. Only fullscreen poster will be used.

Use the img-tag to change the name of the preview image of the video.

<div id="youtube-background" class="hero-video" data-video-id="_M1A_nQZw0Q">
    <img src="images/hero-youtube-bg.jpg" alt="">
</div>

Video Background

Fibre provides the opportunity to add a video in background(which should be composed of a screenshot jpg, a file .mp4, a file .ogv and a file .webm).

To change the file name and the directory where the video is please use the attributes data-path und data-file. 

<div id="video-background" class="hero-video" data-file="video">
    <img src="images/hero-video-bg.jpg" alt="">
</div>
Video is only enabled for Desktop - videos for mobile and tablets are disabled. Only fullscreen poster will be used. Use the img-tag to change the name of the preview image of the video.

Newsletter Form

Current active subscribe form will save emails to your mailchimp list. If you don't have a Mailchimp account you alternative could use the simple php form above:

Write your email and subject prefix in newsletter-process.php:

and use:

<!-- HTML -->
<form action="php/newsletter-process.php" method="post" id="newsletter-form" class="wrapper-h-xs-md">

<!-- instead of
<form id="mc-form"> -->

In order to use the respective Mailchimp subscription form you have to replace the mailchimp post url in the url attribute in custom.js.

// Javascript
$('#mc-form').ajaxChimp({
  ...

  // Replace the URL above with your mailchimp URL (put your URL inside '').
  url: 'http://please enter your valid url to mailchimp'
});
How to find the mailchimp post url?
  1. Login to your mailchimp account
  2. Go to your list
  3. Select Signup Forms
  4. Select Embedded forms
  5. Copy the link and paste into action attribute. (see image below)
