StampReady Integration

You can edit this template in free Stampready builder. All you need is register a free stampready account!

Attention:   You can use Stampready builder for free! Just make a free account and ignore messages about their pricing plans!

Structure

You will find StampReady-integrated template in MainFiles/StampReady folder.

Template named as /TemplateName.zip. You can upload only this file into Stampready builder. If for some reason fol­der doesn't contain this file, and contain just /TemplateName folder, just .zip this folder and then upload to stampready! Another folders in /MainFiles folder created for another editors!

Uploading to StampReady

Login or register in Stampready. Go to Campaigns > New Campaign and click Import template > zip file.

Browse template from MainFiles/StampReady/ folder. When the template loads, click Use template and then paste template name.

Customisation

In this editor you can darag and drop sections, change background colors, background-images, images, titles, text, links, buttons, borders, etc. Also you can send test email or export template for further use in other email clients.

Attention:   Always test your campaigns!

Known bugs

Attention:   If you have bugs with builder, first of all try to disable browser plugins, such us AdBlock and Grammarly! This plugins may have conflict with builder!
