MailChimp

Mailchimp Integration

You can edit this template in Mailchimp editor.

Structure

You will find MailChimp-integrated template in MainFiles/Mailchimp/Template folder.

Template named as /TemplateName.html. You can upload only this file into Mailchimp editor.

Uploading to Mailchimp

Login or register in Mailchimp. Go to Templates > Create Template > Code your own. Then choose Import html. Paste name of your template and browse template from MainFiles/MailChimp/Template folder.

Attention:   After import finished, don't try to edit template in "Template" section. All you ned is click "Save and exit"!

Go to Campaigns > Create Campaign > Create an email. Paste name of your campaign and click "Begin". Then add or create recipient list and click "Next" in right-bottom corner. After that, fill all fields > click "Next". Go to Saved templates and choose previously uploaded template.

Attention:   In editor you will see only FIRST SECTION, please DON'T PANIC. Hover mouse on it and you will see pop-up button in left-bottom corner. Click on "+" and first section would be duplicated. Then hover mouse on second section. From button drop-down menu choose section name that you need! Thats how mailchimp editor works! You cann add "+", delete "-" or choose another section from drop-down menu.

Attention:   Sometimes mailchimp have slow and clunky interface. It's not our fault, and we are not responsible for this! If you have bugs, first of all try to reload browser page, you changes would be saved!

After customization you should click to Preview and Test and test template. You can see mobile view or full page view. Also, you can send test email! You will see how your campaign looks!

Attention:   Sometimes, "Save as template" function don't work. Again, we can't fix that bug, we only make "compatible mailchimp templates" and we are not responsible for mailchimp editor!

Click Next and send template.

Customisation

In this editor you can add or replace sections, change background colors, background images, images, titles, text, links, buttons, borders, fonts, etc. Also you can send test email.

In editor, in right side you will see tabs with different names. Each tab is corresponds to template section with the same name! Find needed tab, click on it and make changes that you need.
For editin images, text, buttons text, icon - just hover and then click on it!
