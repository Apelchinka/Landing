Font changing

You can change font in email template. But you can do it only through code editing.

Attention:   You can use only Google fonts. Also you should change font before uploading to editor.

Instructions

  • Choose template that you need from /Main files/. It could be /Stampready/TemplateName.zip, /Mailchimp/Template/TemplateName.html, /Mailchimp/OutlookSupport/TemplateName.html, /Mailchimp/StampreadyToMailchimp.zip or /CampaignMonitor/index.html
  • If you choose /Stampready/TemplateName.zip or /Mailchimp/StampreadyToMailchimp.zip, you should firstly unzip this files
  • Open index.html (or TemplateName.html) file from unzipped folder or from another chosen folder. For code editing better to use free Sublime Text editor
  • Find links to the old fonts and delete them. In template can be 1 or 2 fonts. Also, there are 2 types of links:

    Fisrt: 
    <link href="https://fonts.googleapis.com/css?family=Hind:400,500|Montserrat:400,500,600,700,900" rel="stylesheet">
    Second: 
    @import url('https://fonts.googleapis.com/css?family=Hind:400,500|Montserrat:400,500,600,700,900');
    Screenshot:
  • Find fonts that you need in Google fonts and copy first links from STANDART and second from @IMPORT.
    • Screenshot:
  • Paste it instead deleted template fonts.
  • Then, in Sublime text click CTRL + F (Find function) and find "font-family". Then highlight all style, for example:
    font-family: 'Hind', Arial, Helvetica, sans-serif;
  • Click ALT + F3 (highlight all styles in document) and change font name inside quotes''. Here is an example for Roboto font, you should get:
    • font-family: 'Roboto', Arial, Helvetica, sans-serif;
  • If template using more then 1 font, then find second and do the same.
  • Save template
  • If you edit unzipped files, that zip it again!
  • Upload to editor.
